A few usage examples:

[geo country="US"]Hello world![/geo]SINGLE COUNTRY
This will display Hello World! to all visitors from USA. Others will not see anything at all.
[geo country="CL,ES,MX"]¡Hola mundo![/geo]MULTIPLE COUNTRIES
This will display ¡Hola mundo! to all visitors from Chile, Spain and Mexico. Others will not see anything at all.

You can also use shortcodes within geo shortcode:

[geo country="CA"]We're ready to go! See details below: [my-other-shortcode][/geo]SHORTCODE-CEPTION
This will display the content to all visitors from Canada and will process [my-other-shortcode] shortcode. Others will not see anything at all.

Coding

The following static functions can be used anywhere along GeoIPLibrary class:

get_client_address()string
Returns the current client's IP address. It bypasses proxies, forwarding and network masks.
get_client_country_code(string $ip = '')string
Returns the current client's ISO 3166-1 alpha-2 country code or the specified in $ip (inherits get_client_address() algorithm in that case).
get_client_country_name(string $ip = '')string
Returns the current client's country name or the specified in $ip (inherits get_client_address() algorithm in that case).
