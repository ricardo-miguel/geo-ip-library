About the library

Geo IP Library (as the name suggests) uses a standalone Geolocation IP library, but it comes from a third-party source. In simple words, this plugin includes a single PHP file that will let you determine the country of location for any given IP address without depending of external web services and PHP dependencies.

The remote source is automatically updated once a day by downloading the free IP address database from Software77.net; so, you can easily update your local library from the source by pressing [ UPDATE NOW] at Geo IP Library's admin page.

Update process consists in downloading and decompressing an updated library file from: http://chir.ag/projects/geoiploc/autogen/geoiploc.tar.gz, then replacing the current local one.

The source project itself is named "php geoiplocation library - ip address to country" and was made by Chirag Mehta. For more information, see below:
